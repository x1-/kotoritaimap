# Kotoritai Map

This project is Real-Time Map.
The map shows Game Centres which have Kotoritai UFO catchers.

## Usage

Start services with sbt:

```
$ sbt
> ~re-start
```

### Testing

Execute tests using `test` command:

```
$ sbt
> test
```
